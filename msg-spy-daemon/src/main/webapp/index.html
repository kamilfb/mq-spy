Welcome to mq-spy-daemon!

You can now use HTTP GET requests to issue control commands to the daemon.

Visit https://github.com/kamilfb/mq-spy/wiki for more information.

The available commands are:

CommandDetailsParametersExample
setScriptsLocation Sets the location where all scripts are stored - otherwise use full paths for all scripts. directory /setScriptsLocation?directory=/mq-spy-daemon/scripts
runScript Runs the given script. name /runScript?name=publish_time.js
runScript Runs the given script; passes all parameters as the 'args' map. name, async, parameters /runScript?name=publish_time.js&async=true&custom_param=test_value
runScriptFunction Runs the given script's function; passes all parameters as the 'args' map. name, parameters /runScriptFunction?name=publish_time_on_request.js&function=publishTime
stopScript Stops the given script. name /stopScript?name=publish_time.js
stop Stops mq-spy-daemon. /stop

Notes: the name parameter is the full name of the script, including the path (unless already set).
